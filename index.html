De Psd a Web Dinámica.

Este es un proceso sencillo de como pasar un archivo .psd a una página web dinámica utilizando las tecnologías web HTML CSS JavaScript, entre otras

Configurar el entorno de desarrollo

Para realizar este ejercicio necesitaremos las siguientes herramientas.

  • Una cuenta registada en GitHub.com
  • Google Chrome Browser (Pues este por defecto viene con CDT (Crome Developers Tools).)
  • Sublime Text 3 (o cualquier editor de Código)

    • Instrucciones

    Ir a la página de SublimeText3

  • Git

    • Instrucciones

    Ir a la página de Git y descargar su version de sistema operativo. en este caso usando linux usaremos el siguiente comando

    $ sudo apt

  • NodeJs (Para poder instalar componentes usando npm (Node Package Manager))
  • Browser Sync
